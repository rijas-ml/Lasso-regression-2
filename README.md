Lasso Regression with Scikit-learn

A simple project demonstrating Lasso Regression (L1 Regularization) using Scikit-learn.

Overview
This mini project uses dummy house price data to show how Lasso Regression works.
It compares predicted and actual data using a simple plot and evaluates performance using the R² score.

 Concepts Covered
 Lasso Regression (L1 Regularization)
 Train-test split
 R² score evaluation
 Visualization using Matplotlib



 Libraries Used
bash
numpy
matplotlib
scikit-learn
